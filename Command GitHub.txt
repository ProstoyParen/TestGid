Вывод конфиг настроек git config --list - вывод всех настроек



  git status - выводит информацию состояния, всегда используеться для проверок команд
  Добавляем файлы пон наблюдения ГИТА командой git add . - Точка значит добавляем все файлы проекта или же нужно писать названия файла которые ставяться под наблюдения (Без точек)


  Для добавления файлов в локальный репозиторий используем команду git commet -a -m"Названия точки сохранения" где -а добавит все файлы -m Добавить коментарий  

  Добавлений/Изменения:
  1. git add . // Добавляем файл на регистрацию изменений 
  2. git commit -m"Название второй точки сохранения " // проведение изменений


  Создаем глобальный репозиторий на сайте https://github.com
  Заходим на сайт в https://github.com/new и создаем новый репозиторий 


  КОМАНДЫ ДОБАВЛЕНИЯ ФАЙЛОВ В ГЛОБАЛЬНЫЙ РЕПОЗИТОРИЙ :
  
  git remode add origin https:github.com/                Соединяет локальный и глобальный репозиторий 
  git push -u origin master Добавляет проект в репозиторий оригинал


  git push -Добавляет в глобальный репозиторий



  Работа со 2 пользователем:
  Создаем папку для проекта, копируем эту папку с помощью команды:
  git clone копируем сылку с сайта (Например: git clone https://github.com/)
  Проект скачан ^-^
  Если 1 пользователь внес изминения и добавил на сервер то 2 пользователь должен зайти в папку проекта и выполнить 2 команды:
  git fetch - извлекает данные из сервера и добавляем в лок репозиторий 2 пользователя но они ене будут видны
  git pull -- показывает изменение 2 пользователя
